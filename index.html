Sample Issue Format

Keep track of how many issues you file!


  ## Issue Description
  // What is the nature of the issue? Which browser and/or AT? How does it impact users?

  ## Current Behavior
  // If the current behavior is a bug, please provide the steps to
  // reproduce. If possible, provide a minimal demo of the problem via 
  // Codepen, JSbin or similar. 

  ## Expected behavior

  ## Additional information
  // What is the motivation / use case for changing the behavior?
  // Stacktraces, related issues, suggestions how to fix
